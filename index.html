Planet's Position

Android app that calculates the position and times of the planets, lunar and solar eclipses, and lunar occultations in the sky.

Download this project as a .zip file Download this project as a tar.gz file

Installation

  1. Install Android Studio (link)
  2. Download the project form the link on the right side of this page.
  3. Import the project into Android Studio.

The Android NDK is required to build this project. Download it here.

Google Maps

This project requires a Google Maps key.

Create a file google_maps_api.xml in one or more of the following locations:

  • /app/src/release/res/values/google_maps_api.xml
  • /app/src/debug/res/values/google_maps_api.xml
  • /app/src/beta/res/values/google_maps_api.xml

The file should contain the following:

<resources>
    <string name="google_maps_key" templateMergeStrategy="preserve" translatable="false">
        [YOUR KEY HERE]
    </string>
</resources>
